# notes-boost

Personal learning notebook for [rwxrob's Beginner Boost](https://twitch.tv/rwxrob) — a live-streamed coding curriculum on Twitch.

## What's here

- **Notes** — searchable references from each session
- **Code** — sample programs written during or after streams
- **Exercises** — experiments and practice problems

## Structure

Subdirectories are named after topics or sessions as they're added.

## Setup

Requires [mise](https://mise.jdx.dev/) and [DevPod](https://devpod.sh/).

```sh
mise install
```
